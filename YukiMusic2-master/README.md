# YukiMusic2.0

> 基于Vue的PC端网页音乐播放器

## 功能介绍
+ 完整的播放器功能
+ 支持多类别搜索
+ 支持收藏
+ 支持MV播放
+ 热门歌手，歌单等推荐功能

+ element-ui
+ stylus
+ npm
+ webpack

## 安装
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
